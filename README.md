# Каталог пищевых добавок E-коды

Веб-приложение для поиска и каталогизации пищевых добавок (E-коды).

## Функциональность

- Просмотр каталога пищевых добавок
- Поиск по коду или названию
- Фильтрация по категориям и статусу безопасности
- Регистрация и авторизация пользователей
- Добавление добавок в избранное
- CRUD операции для зарегистрированных пользователей
- REST API
